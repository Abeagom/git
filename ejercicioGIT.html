Examen Git y Github

Alumn@: A.B.G.

  1. ¿Que es un sistema de control de versiones?

    Un sistema que permite volver a versiones anteriores en caso de no compatibilidad o fallo.

  2. ¿Cual es la diferencia entre git y github?

    Github es el sitio donde publicar repositorios con código y git es la herramienta con la que trabajar con ellos.

  3. ¿Cómo creo una rama en git?

    git branch [nombre-rama]

  4. ¿Cómo clono un repositorio?

    git clone [url-repositorio]

  5. ¿Cómo puedo bajar los ultimos cambios de un repositorio?

    git pull [alias][rama]

  6. ¿Cómo puedo gener un tag?

    git tag [nombre-rama]

  7. Cómo hago una mezcla de ramas?

    git merge [nombre-rama]
